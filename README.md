
# Hangabubu 🐰

A Pygame-based Hangman game featuring custom **Labubu** sprites instead of the traditional stickman — set against a desert backdrop with hand-drawn spritework and a fully custom UI.
This was a project demonstration of data structures for CMSC 141 in UP Cebu, 2025.

---

## 🎮 Gameplay

- Guess the hidden word one letter at a time
- Each wrong guess progresses the Labubu sprite through its hanging animation
- Words come with category hints to help you guess
- Multiple difficulty levels

---

| Layer | Technology |
|-------|-----------|
| Language | Python |
| Framework | Pygame |
| Art | Custom hand-drawn sprites |

---

## 🚀 Running the Game

1. Make sure Python and Pygame are installed:
   ```bash
   pip install pygame
   ```
2. Clone the repo and run:
   ```bash
   git clone https://github.com/ChewyFishLegs/Hangabubu.git
   cd Hangabubu
   python main.py
   ```
